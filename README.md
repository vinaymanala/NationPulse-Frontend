## Frontend - Nation Pulse Project

A platform to explore and learn about several countries demographics and economy. The project uses Typescript with React and uses React query for handling API calls, caching data, refetch, etc.

### Local Development

To run the project locally, make sure you have Nodejs latest version installed. You would need make tool to run this command:

```
make start
```

That's it! You frontend should be running now. If still there are issues. Just let me or Google/AI know. They can help definitely

To run the project backend, you can follow the instructions [here](https://github.com/vinaymanala/NationPulse)
